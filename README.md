# Time Tracker Go

🚧 **開発中 / Under Development** 🚧

軽量で高速なmacOS向けタイムトラッキングデスクトップアプリケーション

## ステータス

このプロジェクトは現在活発に開発中です。MVP（Minimum Viable Product）の完成に向けて作業を進めています。

### 技術スタック

- **フレームワーク**: Tauri 2.x
- **バックエンド**: Rust
- **フロントエンド**: React + TypeScript
- **データベース**: SQLite
- **開発手法**: DDD + TDD

### 開発ブランチ

実際の開発作業は `develop` ブランチで行われています。

```bash
git checkout develop
```

### 予定機能

- [ ] プロジェクト/タスク管理
- [ ] タイムトラッキング
- [ ] レポート生成
- [ ] タグ機能
